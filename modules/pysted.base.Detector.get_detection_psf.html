Detector.get_detection_psf(lambda_, psf, na, transmission, datamap_pixelsize)ÔÉÅ

Compute the detection PSF as a convolution between the fluorscence PSF and a pinhole, as described by the equation from [Willig2006].

The pinhole radius is determined using the n_airy, the fluorescence wavelength, and the numerical aperture of the objective.

Parameters:

  • lambda ‚Äì The fluorescence wavelength (m).

  • psf ‚Äì The fluorescence PSF that can the obtained using get_psf().

  • na ‚Äì The numerical aperture of the objective.

  • transmission ‚Äì The transmission ratio of the objective for the given fluorescence wavelength lambda_.

  • datamap_pixelsize ‚Äì The size of a pixel in the simulated image (m).

Returns:

A 2D array.
